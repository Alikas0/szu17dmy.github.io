    11 Java网络编程

    11.1 网络地址

    InetAddress类中的（ ）方法可以实现正向名称解析。

    A. isReachable

    B. getHostAddress()

    C. getHostName()

    D. getByName()

    解决方案

    D

    11.2 UDP数据报

    使用UDP套接字通信时，常用（ ）类把要发送的信息打包。

    A. String

    B. DatagramSocket

    C. MulticastSocket

    D. DatagramPacket

    解决方案

    D

    11.3 TCP连接

    ServerSocket的监听方法accept()的返回值类型是（ ）。

    A. void

    B. Object

    C. Socket

    D. DatagramPacket

    解决方案

    C
