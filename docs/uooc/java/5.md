## 5.1 String类
### 编译及运行以下代码，下列选项哪个是正确的？
``` java
String s=new String("Bicycle");
int iBegin=1;
char iEnd=3;
System.out.println(s.substring(iBegin,iEnd));
```

B. 输出ic

C. 输出icy
